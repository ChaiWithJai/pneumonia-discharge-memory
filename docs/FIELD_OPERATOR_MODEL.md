# Field Operator Model

## Champion

The champion is a pulmonary service line leader or Chief Clinical Officer responsible for discharge quality, readmission pressure, throughput, and clinician time.

## Story

The buyer does not need another dashboard. They need a way to turn chart context into governed action.

```text
Pneumonia patient population
  -> service-line question
  -> patient-level reasoning trace
  -> what-if discharge scenarios
  -> structured human handoff
  -> institutional memory
```

## Talk Track

"We are not asking a model to decide discharge. We are building a governed workflow that helps the service line see the risk structure, inspect the reasoning, stress-test assumptions, and hand the case to a clinician with the right context."

## Worked Example

Patient is recovering from pneumonia but has:

- Afebrile duration under 48 hours.
- Borderline inflammatory trajectory.
- Mobility decline.
- Medication access risk.
- Limited home support.

The system should not simply say "high risk." It should show:

- Which instruments fired.
- Which data points mattered.
- Which assumptions change the risk.
- Which operational supports could change the discharge path.
- What must be reviewed by a clinician.
