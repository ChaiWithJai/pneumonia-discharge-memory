# Contributing

Contributions should improve the reference architecture, documentation, tests, or hackathon usability.

## Ground Rules

- Use synthetic data only.
- Do not add PHI, screenshots from real charts, or identifiable patient stories.
- Keep generated text and generated images non-authoritative.
- Add tests for scoring or state-machine changes.
- Document clinical assumptions and source basis.

## Useful Contribution Areas

- Additional synthetic pneumonia cases.
- COPD or asthma reuse examples that prove institutional memory.
- Better what-if scenario visualization.
- Local model adapter improvements.
- Evaluation harnesses for correctness, bias, toxicity, and handoff quality.
