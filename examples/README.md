# Examples

## `patients/pneumonia_case_001.json`

Synthetic high-risk pneumonia discharge case.

Use it to demonstrate:

- Frailty risk.
- Infection trend concern.
- Medication access risk.
- What-if scenarios.
- Structured human handoff.
- Institutional memory persistence.

## `pneumonia_case_001_output.json`

Generated output from:

```bash
pdm-run examples/patients/pneumonia_case_001.json --json
```

## `memory/institutional_memory.jsonl`

Append-only local memory events from CLI runs.
